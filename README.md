## Project Description

This project predicts student academic performance using machine learning techniques. It analyzes both academic and behavioral factors such as marks, sleep hours, stress level, and social media usage.

The system helps identify whether a student is likely to perform **well or poorly**, enabling early intervention and better academic support.

---

## Features Used

- Math Score
- Reading Score
- Writing Score
- Sleep Hours
- Stress Level
- Social Media Usage
- Average Score

---

## Machine Learning Models

The following models were implemented and compared:

- Logistic Regression
- Decision Tree
- Random Forest
- K-Nearest Neighbors (KNN)

**Best Model:** Random Forest (Highest Accuracy)
